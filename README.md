# 地理知识学习 Skill 项目仓库
## 📌 项目介绍
本仓库基于 Kimi Code 开发，包含地理知识学习专属 Skill 的完整源码、可分发安装包与创建全流程对话记录，覆盖自然地理、人文地理、区域地理三大核心模块，适用于地理学习、考试解题、区域分析等场景。

## ✨ 核心文件说明
| 文件/文件夹 | 作用 |
|------|------|
| `geography-learning/` | Skill 源码目录，核心为 `SKILL.md` 配置文件 |
| `geography-learning.skill` | 可直接导入 Kimi Code 的 Skill 安装包 |
| `geography-skill-conversation.md` | Skill 从创建到打包的完整对话记录 |

## 🚀 使用方法
1.  下载 `geography-learning.skill` 文件
2.  在 Kimi Code 中执行 `/install geography-learning.skill`
3.  输入 `@geography-learning` 即可激活使用

## 🧩 核心能力
- 自然地理：地球运动、气候类型、水循环、地貌分析
- 人文地理：区位分析、城市化、产业转移、可持续发展
- 区域地理：区域对比、综合题解题、地图读图析图
